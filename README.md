Research Assignment 1 – README
Overview

This assignment is part of the Foundations of Data Analytics & Data Science course. It covers key concepts in data, artificial intelligence, machine learning, APIs, cloud computing, and data ethics. The purpose of this assignment is to build a strong foundational understanding of how data and AI are used in real-world applications.

Objectives
Understand core data concepts such as databases, data warehouses, and data lakes
Explore Artificial Intelligence (AI), Machine Learning (ML), and Deep Learning
Learn about APIs, cloud computing, and developer tools
Develop awareness of data ethics, governance, and privacy
Apply research skills and explain concepts in your own words
Assignment Structure

The assignment is divided into five sections:

Section A: Data Fundamentals

Focuses on databases, data storage, data modeling, ETL processes, and data formats.

Section B: Artificial Intelligence & Machine Learning

Covers AI concepts, ML types, deep learning, NLP, and computer vision.

Section C: APIs, Tools & Developer Concepts

Introduces APIs, JSON, cloud computing, version control, Python, and Jupyter Notebooks.

Section D: Data Ethics, Governance & Careers

Explores data privacy, ethics, governance, bias in AI, and career roles in data.

Section E: Personal Reflection

Encourages critical thinking about learning experiences and future applications of data and AI.
